🧪 How to Use

1. Open the app in your browser
2. Type any domain in the input field (e.g. google.com, example.com)
3. Click ⚡ SCAN TARGET or press Enter
4. Watch the real-time terminal log and progress steps
5. View results: Risk score, charts, AI analysis, predictions
6. Download your report using the export buttons
